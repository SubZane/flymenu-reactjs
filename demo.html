About flymenu-reactjs

flymenu-reactjs is an off-canvas menu component written in ReactJS with TypeScript and Styled Components.

Getting started

Go to the Github page and follow instructions

Features

  • Populated by JSON-data
  • Customizable with diferent types of animation effects
  • Extendable. Put anything in the off-canvas panel instead of the menu (if you wish)

Demonstration

Use the round button in the bottom right corner to open up the menu
